# 🕉️ IGS Convention Centre — Booking Management System

**Tirunelveli's Premier Convention Venue**

A complete hall booking management system for IGS Convention Centre, Palayamkottai, Tirunelveli.

## Features
- New Booking with auto rent population
- Modify / Edit existing bookings
- Cancel bookings with refund calculation
- Invoice, Advance Receipt & Quotation printing
- Monthly Calendar view
- Guest booking history
- Pricing reference
- Data saved in browser localStorage
